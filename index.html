    Headings

    Headings are mainly used to break up the text, but they also have a significant impact on SEO. It is important to follow the correct heading hierarchy, because it not only makes it easier for people to read, but also helps search engines understand the structure of the content properly. Properly optimized headlines can improve a website's search engine visibility and increase website traffic.

    Styled text

    In the text you can use tag <em> for italic text (also can be used tag <i>, but the tag defines a part of text in an alternate voice or mood. It is mainly just for term or a phrase from another language!). Tags <strong> or <b> are used for bold text and <mark> for highlighted text. Tag <u> is for underlined text and tag <small> is for smaller text. Here is striked text by tag <s>. For subscript and superscript text are used <sub> and <sup> tags. Tag <abbr> is for abbreviation – example: NASA. Text can be colored using a class within the <span> tag.

    Lists

    Unordered list

    The <ul> tag is used to create an unordered list. Combine it with the <li> tag to define the list items within the unordered list.

    • Shark
    • Jellyfish
    • Fish
      • Anemonefish
      • Whale
    • Porcupinefish

    Ordered list

    The <ol> tag is used to create an ordered list. Combine it with the <li> tag to define the list items within the ordered list.

    1. Whale
    2. Shark
    3. Fish
      1. Anemonefish
      2. Porcupinefish
    4. Jellyfish
